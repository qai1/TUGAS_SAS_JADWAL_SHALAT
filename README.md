# TUGAS_SAS_JADWAL_SHALAT

> Proyek untuk menampilkan jadwal shalat berdasarkan lokasi — tugas mata kuliah SAS / Komputasi Astronomi / semacamnya.

## Deskripsi  
Aplikasi ini menampilkan jadwal shalat harian (dan/atau bulanan) berdasarkan lokasi/kota yang dipilih.  
Proyek ini dibangun menggunakan React / TypeScript (sesuai struktur `app/`, `components/`, `types/`), sehingga bisa dijalankan di berbagai platform dengan antarmuka modern.

## Fitur  
- Menampilkan waktu shalat untuk lokasi tertentu (misalnya kota pengguna)  
- Kemungkinan untuk mengganti lokasi — sehingga jadwal bisa sesuai posisi geografis  
- Antarmuka user-friendly berbasis web / mobile (jika menggunakan framework sesuai)  
- (Opsional) Menyimpan preferensi lokasi/perhitungan — tergantung implementasi

## Instalasi & Jalankan (Development)  
1. Clone repository  
   git clone https://github.com/qai1/TUGAS_SAS_JADWAL_SHALAT.git
   cd TUGAS_SAS_JADWAL_SHALAT
2. Install dependensi
   npm install
3. Jalankan Aplikasi
   npx expo start
